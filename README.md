# outdated
This program converts dates from the U.S. format (MM/DD/YYYY or “Month DD, YYYY”) into the international ISO 8601 format (YYYY-MM-DD). Users can input either numeric or textual dates (e.g., 9/8/1636 or September 8, 1636), and the program outputs the properly padded, sortable version. Invalid inputs are rejected until a valid date is entered.
